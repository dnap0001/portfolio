# Portfolio

A plain HTML/CSS portfolio website, ready for GitHub Pages.

## Pages

- **Home** — Landing page with intro
- **Work** — 3–4 project showcases
- **Contact** — Contact info and links
- **Resume** — HTML resume + PDF download

## Deploy to GitHub Pages

1. Create a new repo on GitHub (e.g. `portfolio` or `username.github.io`)
2. Push this project to the repo
3. Go to **Settings → Pages**
4. Under **Source**, choose **Deploy from a branch**
5. Select branch `main` and folder `/ (root)`
6. Save — your site will be at `https://<username>.github.io/<repo-name>`

## Customize

Replace placeholder content with your own:

- **index.html** — Name, tagline, bio
- **work.html** — Project titles, descriptions, images (add to `images/`), links
- **contact.html** — Email, LinkedIn, GitHub URLs
- **resume.html** — Resume content in HTML
- **resume.pdf** — Add your PDF file to the project root for the Download button

To add a contact form, sign up at [Formspree](https://formspree.io), create a form, and uncomment the form block in `contact.html` with your form ID.
